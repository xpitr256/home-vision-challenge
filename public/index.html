Checkbox Detection by Pablo Rodriguez Massuh

Welcome to the Checkbox Detection API. This API is designed to detect checkboxes in images and determine whether they are checked or unchecked.

How It Works

This API allows you to send an image either through a GET request for predefined test images or via a POST request for uploaded images, returning structured data with the location and status of detected checkboxes.

Usage

Send a request to the /checkbox endpoint:

GET Example

GET /checkbox?size=24

This uses a predefined test image.

POST Example

POST /checkbox
Content-Type: multipart/form-data

--boundary
Content-Disposition: form-data; name="image"; filename="your-image.jpg"
Content-Type: image/jpeg

[image data here]
--boundary--

This uploads an image from your device. You can also include the size parameter in the query string, e.g., /checkbox?size=24.

Response Format

{
  "image_name": "test-image.jpg" or "uploaded-image.jpg",
  "total_detections": 42,
  "checkbox_size_in_pixels": 24,
  "image_with_checkboxes_url": "/response/image_with_checkboxes.jpg",
  "checkboxes": [
    { "x": 156, "y": 96, "status": "unchecked" },
    { "x": 282, "y": 96, "status": "checked" }
  ]
}
